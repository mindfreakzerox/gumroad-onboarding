# VA SOP — Fulfillment & Support

Purpose: run day-to-day fulfillment for the Gumroad product.

Tasks
1. Monitor sales
   - Check gumroad_events.log or the Google Sheet for new rows
   - For each sale: send Welcome Email (use template in marketing/email_templates.md)
2. Assist buyers
   - If buyer can't import Zapier recipe, follow the Quickstart checklist
   - If technical, escalate to owner (tag on Slack/Discord)
3. Social posting
   - Use the /marketing/posts.md file. Post per schedule and repost top performing posts.
4. Refunds
   - Follow refund policy in README. Issue refunds via Gumroad admin and log reason.

Tools
- Gumroad admin
- Slack/Discord for escalations
- Google Sheets for tracking

Contact owner for any unusual issues.
